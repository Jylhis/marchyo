---
title: Keybindings & Navigation
description: The Hyprland hotkeys shipped with the Marchyo desktop
---

The Marchyo desktop is keyboard-driven. Enabling `marchyo.desktop.enable` ships
Hyprland with the keybindings below, plus the mako notification daemon, the
vicinae launcher, cliphist clipboard history, and grimblast/satty/tesseract
screenshots. `Super` is the primary modifier (the Windows/Command key).

:::note
These binds are declared across several Home Manager modules — the bulk in
`modules/home/hyprland.nix`, with the rest contributed by `screenshot.nix`,
`omarchy-binds.nix`, `menus.nix`, `utilities.nix`, `window-toggles.nix`,
`keybindings-cheatsheet.nix`, and `webapps.nix` — which together are the source
of truth. To change one, override the Home Manager
`wayland.windowManager.hyprland.settings` in your own config rather than
editing a dotfile at runtime.
:::

## Launching apps

| Keys | Action |
|------|--------|
| `Super + Return` | Terminal (Ghostty) |
| `Super + B` | Web browser |
| `Super + F` | File manager |
| `Super + R` | App launcher (vicinae) |
| `Super + M` | Music player |
| `Super + E` | Editor |
| `Super + O` | Obsidian |
| `Super + G` | Messenger |
| `Super + /` | Password manager (focuses 1Password if open) |
| `Super + A` | AI chat (only when `marchyo.ai.enable` is on and `marchyo.ai.tooling.enable` — the default — isn't turned off) |
| `Super + .` | Emoji picker (vicinae) |
| `Super + Shift + I` | Input-method config (fcitx5) |

### Web apps

With `marchyo.webapps.enable` (on by default with the desktop), a set of sites
opens as standalone browser app windows on `Super + Shift` chords:

| Keys | Action |
|------|--------|
| `Super + Shift + A` | ChatGPT |
| `Super + Shift + G` | GitHub |
| `Super + Shift + Y` | YouTube |
| `Super + Shift + W` | WhatsApp |
| `Super + Shift + Z` | Zoom |
| `Super + Shift + X` | X |
| `Super + Shift + P` | Google Photos |

Discord, Google Calendar, and Gmail are registered as launchers but ship no
default key — start them from the app launcher, or give them one via
`marchyo.webapps.apps`.

## Workspaces & scratchpad

| Keys | Action |
|------|--------|
| `Super + 1…5` | Switch to workspace (on the current monitor) |
| `Super + Shift + 1…0` | Move active window to workspace 1–10 |
| `Super + Shift + Alt + 1…0` | Move window there silently (don't follow) |
| `Super + Tab` / `Super + Shift + Tab` | Next / previous workspace |
| `Super + Ctrl + Tab` | Former workspace |
| `Super + scroll` | Scroll through workspaces |
| `Super + D` | Toggle the Drawer (scratchpad workspace) |
| `Super + Shift + D` | Send active window to the Drawer |

## Multi-monitor

| Keys | Action |
|------|--------|
| `Ctrl + Alt + Tab` / `Ctrl + Alt + Shift + Tab` | Focus next / previous monitor |
| `Super + Shift + ,` / `Super + Shift + .` | Move active window to the adjacent monitor |
| `Super + Shift + Alt + ←/→/↑/↓` | Move the whole workspace to an adjacent monitor |

## Screenshots, clipboard & OCR

| Keys | Action |
|------|--------|
| `Super + S` (or `Print`) | Screenshot a region/window (copy + save) |
| `Super + Shift + S` (or `Super + Shift + Print`) | Screenshot a region and annotate (satty) |
| `Super + Ctrl + S` (or `Shift + Print`) | Screenshot the active window |
| `Super + Alt + S` (or `Ctrl + Print`) | Screenshot the current screen |
| `Super + Ctrl + Alt + S` (or `Alt + Print`) | Screenshot all screens |
| `Super + Shift + O` (or `Super + Ctrl + Shift + S`) | OCR a region → text to clipboard (tesseract) |
| `Super + Ctrl + V` | Clipboard history (vicinae/cliphist) |
| `Super + Shift + C` | Pick a color (hex to clipboard) |

Screenshots land in `~/Pictures/Screenshots`. Annotation and OCR can be turned
off with `marchyo.screenshot.enableAnnotation` / `enableOcr`.

## Top bar (Waybar) mouse actions

Segments that open a tool launch it as a centered floating popup — never as a
tile in the workspace. Scrolling and right-clicks act in place.

| Segment | Action |
|---------|--------|
| Workspace numbers | Click to switch workspace |
| Clock | Click to toggle the date/week-number format |
| `·` (left of the tray) | Hover to expand the system tray |
| Bell | Click to toggle notification do-not-disturb |
| Mic/dictation (if enabled) | Left-click toggles recording; right-click opens the floating status window |
| Layout code (`en`, `fi`, …) | Click to cycle keyboard layouts |
| `bt` | Click for the Bluetooth manager (bluetui, floating) |
| Wi-Fi / `eth` | Click for the Wi-Fi manager (impala, floating) |
| `vol` | Click for the audio mixer (wiremix, floating); right-click mutes; scroll changes volume |
| `cpu` | Click for the system monitor (btop, floating) |
| `eco`/`bal`/`perf` | Click to cycle the power profile (built into Waybar) |
| `bat`/`chg`/`pwr` | Click for the power/session menu (floating) |
