Quick Start

1

Add to Your Flake

{
  inputs = {
    nixpkgs.url = "github:NixOS/nixpkgs/nixos-unstable";
    marchyo.url = "github:Jylhis/marchyo";
  };
}
2

Import Modules

nixosConfigurations.myhost = nixpkgs.lib.nixosSystem {
  modules = [
    marchyo.nixosModules.default
    ./configuration.nix
  ];
};
3

Configure

marchyo = {
  desktop.enable = true;
  development.enable = true;

  users.alice = {
    fullname = "Alice Smith";
    email = "alice@example.com";
  };
};
4

Build & Enjoy

# Build your system
sudo nixos-rebuild switch --flake .#myhost

# Start using Marchyo!

What's Included

Desktop Environment

  • Hyprland (Wayland compositor)
  • Waybar (status bar)
  • Vicinae (launcher)
  • Mako (notifications)
  • Kitty (terminal)

Development Tools

  • Docker & libvirtd
  • Git with LFS
  • Direnv & nix-direnv
  • LazyGit, LazyDocker
  • Build tools (gcc, make, cmake)

TUI Applications

  • btop (system monitor)
  • k9s (Kubernetes)
  • bluetui (Bluetooth)
  • sysz (systemd)
  • fastfetch (system info)

Modern CLI Tools

  • ripgrep, fd (search)
  • eza, bat (file operations)
  • fzf (fuzzy finder)
  • duf, gping (utilities)
  • aria2, xh (network)
